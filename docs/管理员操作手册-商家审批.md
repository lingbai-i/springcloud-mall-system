## 📋 功能概述

商家入驻审批系统允许管理员审核商家的入驻申请，包括查看申请详情、审批通过/拒绝、查看审批历史等功能。

## 📂 审批流程

### Step 1: 访问申请列表

1. 登录管理员后台
2. 点击左侧菜单 **"商家管理"** → **"入驻申请"**
3. 进入申请列表页面

### Step 2: 筛选申请

**按状态筛选**:

- 全部状态
- 待审核 ⏳
- 已通过 ✅
- 已拒绝 ❌

**搜索功能**:

- 输入店铺名称、联系人或电话号码
- 按回车键搜索

### Step 3: 查看申请详情

1. 在列表中点击 **"查看"** 按钮
2. 弹出详情对话框，显示：
   - 店铺基本信息
   - 联系人信息
   - 企业/个人证件信息
   - 营业执照/身份证图片（可点击放大）
   - 审批状态和历史

### Step 4: 审批申请

#### 审批通过流程

1. 在列表中找到待审核的申请
2. 点击 **"审批"** 按钮
3. 在审批对话框中：
   - 选择 **"通过"**
   - （可选）填写审批意见
4. 点击 **"确认审批"**
5. 系统将自动：
   - 创建商家账号
   - 发送通过短信
   - 更新申请状态

#### 审批拒绝流程

1. 在列表中找到待审核的申请
2. 点击 **"审批"** 按钮
3. 在审批对话框中：
   - 选择 **"拒绝"**
   - **必须**填写拒绝原因
4. 点击 **"确认审批"**
5. 系统将自动：
   - 发送拒绝短信（包含原因）
   - 更新申请状态

## 🎯 审批注意事项

### ✅ 通过审批时检查

1. **证件真实性**

   - 营业执照/身份证是否清晰可见
   - 信息是否完整准确
   - 有效期是否在有效范围内

2. **信息一致性**

   - 企业名称与营业执照一致
   - 法定代表人信息准确
   - 联系方式真实有效

3. **合规性检查**
   - 经营范围是否合法
   - 是否符合平台入驻条件
   - 无不良经营记录

### ❌ 拒绝审批时注意

1. **必须填写拒绝原因**

   - 原因要具体明确
   - 便于商家了解问题并改进
   - 示例：
     - "营业执照模糊不清，请重新上传"
     - "统一社会信用代码格式错误"
     - "经营范围不符合平台要求"

2. **拒绝原因建议**
   - 清晰说明问题所在
   - 给出改进建议
   - 保持专业礼貌

## 🔍 常见问题

### Q1: 审批后可以撤销吗？

**A**: 当前版本不支持撤销审批。建议审批前仔细核对信息。

### Q2: 商家多久能收到审批通知？

**A**: 审批操作完成后，系统会立即发送短信通知到商家注册的手机号。

### Q3: 审批通过后商家如何登录？

**A**: 商家使用申请时填写的账号密码，访问商家后台登录：

- 地址: http://localhost:5173/merchant/login
- 账号: 申请时填写的 username
- 密码: 申请时填写的 password

### Q4: 如果审批时系统报错怎么办？

**A**:

1. 刷新页面重试
2. 检查网络连接
3. 联系技术支持查看后端日志
4. 常见错误：
   - "该申请已被审批" - 申请已处理过
   - "拒绝时必须填写原因" - 拒绝申请时未填原因

### Q5: 申请列表可以导出吗？

**A**: 当前版本暂不支持导出功能，后续版本将添加。

## 🛡️ 安全提示

1. **保护管理员账号**

   - 定期修改密码
   - 不要共享账号
   - 及时登出系统

2. **谨慎审批操作**

   - 审批前仔细核对信息
   - 审批操作会被完整记录
   - 包括操作人、时间、IP 地址

3. **敏感信息保护**
   - 不要截图分享商家证件信息
   - 不要泄露商家联系方式
   - 遵守数据保护法规
