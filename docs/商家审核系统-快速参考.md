## 📋 审核检查清单

### 企业/个体工商户

- [ ] 企业名称与营业执照一致
- [ ] 统一社会信用代码格式正确（18位）
- [ ] 法定代表人信息准确
- [ ] 营业执照照片清晰可辨认
- [ ] 营业执照在有效期内
- [ ] 联系方式有效
- [ ] 店铺名称合规（无违禁词）

### 个人商家

- [ ] 身份证号格式正确（18位）
- [ ] 身份证照片清晰完整
- [ ] 身份证在有效期内
- [ ] 姓名与身份证一致
- [ ] 联系方式有效
- [ ] 店铺名称合规

## 🎯 审批标准

### 通过条件

✅ 所有必填信息完整  
✅ 证件照片清晰有效  
✅ 信息真实准确  
✅ 店铺名称合规  
✅ 符合平台入驻政策

### 拒绝情况

❌ 证件不清晰或无效  
❌ 信息填写不完整  
❌ 证件已过期  
❌ 信息不一致  
❌ 店铺名称违规  
❌ 重复申请

## 🔧 故障排除

### 问题：页面加载失败

**解决方案**:
1. 检查网络连接
2. 刷新页面（F5）
3. 清除浏览器缓存
4. 检查服务是否正常运行

### 问题：图片无法显示

**解决方案**:
1. 检查图片URL是否有效
2. 检查MinIO服务是否运行（端口9000）
3. 尝试在新标签页打开图片URL

### 问题：审批提交失败

**解决方案**:
1. 检查网络连接
2. 确认审批意见已填写（拒绝时必填）
3. 重新登录后再试
4. 查看浏览器控制台错误信息

### 问题：列表数据不更新

**解决方案**:
1. 点击"刷新"按钮
2. 清除筛选条件
3. 刷新浏览器页面
4. 检查后端服务日志
