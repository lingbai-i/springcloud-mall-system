## 系统概述

### 功能简介

商家审核系统是在线商城管理后台的核心模块之一，负责处理商家入驻申请的审核工作。主要功能包括：

- ✅ 查看商家入驻申请列表
- ✅ 审核商家资质信息
- ✅ 通过或拒绝入驻申请
- ✅ 查看审核历史记录
- ✅ 统计数据分析

### 系统访问

- **访问地址**: http://localhost:5173/admin
- **登录账号**: admin / admin123（请修改默认密码）
- **浏览器要求**: Chrome 90+, Firefox 88+, Edge 90+

## 商家申请管理

### 访问申请管理页面

**方式 1**: 通过导航菜单
- 点击左侧菜单 "商家管理" → "入驻申请"

**方式 2**: 直接访问
- 访问 http://localhost:5173/admin/merchants/applications

### 页面功能说明

#### 1. 统计卡片

页面顶部显示四个统计卡片：

| 卡片 | 说明 | 操作 |
|------|------|------|
| 待审核 | 等待审核的申请数量 | 点击可快速筛选待审核申请 |
| 已通过 | 审核通过的申请数量 | 点击可查看已通过的申请 |
| 已拒绝 | 审核拒绝的申请数量 | 点击可查看已拒绝的申请 |
| 总申请数 | 所有申请的总数 | 点击可查看全部申请 |

**提示**: 点击统计卡片可快速筛选对应状态的申请。

#### 2. 筛选和搜索

**筛选选项**:
- **审批状态**: 待审核 / 已通过 / 已拒绝
- **搜索框**: 支持按店铺名称、联系人、手机号搜索

**操作步骤**:
1. 选择筛选条件
2. 输入搜索关键词
3. 点击"搜索"按钮或按回车键
4. 点击"刷新"按钮重新加载数据

#### 3. 申请列表

列表显示以下信息：

| 列名 | 说明 |
|------|------|
| 申请ID | 系统自动生成的唯一标识 |
| 店铺名称 | 商家申请的店铺名称 |
| 主体类型 | 企业 / 个体工商户 / 个人 |
| 店铺类型 | 旗舰店 / 专卖店 / 专营店等 |
| 联系人 | 商家联系人姓名 |
| 联系电话 | 脱敏显示（138****8000） |
| 申请时间 | 申请提交的时间 |
| 审批状态 | 待审核 / 已通过 / 已拒绝 |
| 操作 | 查看 / 审批按钮 |

**操作说明**:
- **查看**: 查看申请的完整详细信息
- **审批**: 对待审核的申请进行审批（仅待审核状态显示）
- **双击行**: 快速查看申请详情

## 审核操作指南

### 审核前准备

在审核商家申请前，请确认以下信息：

1. ✅ 店铺名称是否合规（无违禁词、无侵权）
2. ✅ 主体信息是否真实完整
3. ✅ 证件照片是否清晰可辨认
4. ✅ 联系方式是否有效
5. ✅ 经营范围是否符合平台规定

### 审核步骤

#### 步骤 1: 打开审批对话框

1. 在申请列表中找到需要审核的申请
2. 点击"审批"按钮
3. 系统打开审批对话框

#### 步骤 2: 查看申请信息

审批对话框显示以下信息：

**基本信息**:
- 店铺名称
- 主体类型和店铺类型
- 联系人和联系方式
- 登录账号
- 申请时间

**企业资质**（企业/个体）:
- 企业名称
- 统一社会信用代码
- 法定代表人
- 营业执照照片（可点击放大查看）

**个人资质**（个人）:
- 身份证号
- 身份证正反面照片（可点击放大查看）

#### 步骤 3: 查看证件照片

**查看方式**:
1. 点击证件图片可放大查看
2. 支持缩放和拖动
3. 可切换查看多张图片

**审核要点**:
- ✅ 照片清晰，信息完整
- ✅ 证件在有效期内
- ✅ 证件信息与填写信息一致
- ✅ 无PS痕迹，无遮挡

#### 步骤 4: 做出审批决定

**通过审核**:
1. 选择"通过审核"
2. （选填）填写审批意见，如"资料齐全，符合入驻条件"
3. 点击"确认通过"按钮
4. 系统确认后自动创建商家账号

**拒绝申请**:
1. 选择"拒绝申请"
2. **必填**填写拒绝原因，至少5个字符
3. 点击"确认拒绝"按钮
4. 系统更新申请状态为已拒绝

#### 步骤 5: 确认操作

系统会弹出二次确认对话框：
- 通过：提示"确认通过 XXX 的入驻申请吗？通过后将自动创建商家账号。"
- 拒绝：提示"确认拒绝 XXX 的入驻申请吗？"

点击"确认"完成审批。

### 审批后效果

**审批通过后**:
- ✅ 申请状态变更为"已通过"
- ✅ 自动创建商家账号（`merchant` 表）
- ✅ 商家可使用申请时的账号密码登录
- ✅ 待审核数量 -1，已通过数量 +1
- ✅ （未来）发送短信通知商家

**审批拒绝后**:
- ✅ 申请状态变更为"已拒绝"
- ✅ 记录拒绝原因
- ✅ 待审核数量 -1，已拒绝数量 +1
- ✅ （未来）发送短信通知商家

## 常见问题处理

### 问题 1: 证件照片看不清

**解决方案**:
1. 点击照片放大查看
2. 如果仍然不清晰，选择"拒绝申请"
3. 填写原因："证件照片不清晰，请重新上传高清照片"

### 问题 2: 营业执照已过期

**解决方案**:
1. 检查营业执照上的有效期
2. 如已过期，选择"拒绝申请"
3. 填写原因："营业执照已过期，请更新后重新申请"

### 问题 3: 信息填写不完整

**解决方案**:
1. 检查必填字段是否都已填写
2. 选择"拒绝申请"
3. 填写原因："XXX信息缺失，请补充完整后重新申请"

### 问题 4: 店铺名称不合规

**解决方案**:
1. 检查店铺名称是否包含违禁词
2. 检查是否侵犯他人商标
3. 选择"拒绝申请"
4. 填写原因："店铺名称不合规：XXX，请修改后重新申请"

### 问题 5: 重复申请

**解决方案**:
1. 搜索该商家的历史申请记录
2. 如果之前已通过，选择"拒绝申请"
3. 填写原因："该商家已有账号，请直接登录使用"

## 最佳实践

### 审核效率

1. **优先处理原则**:
   - 优先审核企业商家（资质更完整）
   - 优先审核信息完整的申请
   - 按申请时间先后顺序处理

2. **批量处理**:
   - 集中时间段处理审核
   - 建议每天固定时间审核（如上午10点、下午3点）
   - 避免申请积压

3. **审核标准**:
   - 制定统一的审核标准文档
   - 对不同主体类型有明确的审核要点
   - 保持审核标准的一致性

### 审批意见规范

**通过时的建议意见**:
- "资料齐全，审核通过"
- "企业资质完整，符合入驻条件"
- "个人信息真实有效，同意入驻"

**拒绝时的建议意见**:
- "营业执照已过期，请更新后重新申请"
- "身份证照片不清晰，请重新上传高清照片"
- "店铺名称包含违禁词'XXX'，请修改后重新申请"
- "联系电话无法接通，请确认后重新提交"
- "企业信息与营业执照不符，请核实后重新申请"

### 数据安全

1. **信息保护**:
   - 不要将商家的证件信息外传
   - 不要截图或拍照保存证件
   - 审核完成后及时关闭页面

2. **账号安全**:
   - 定期修改管理员密码
   - 不要共享管理员账号
   - 离开时及时退出登录

## 功能详细说明

### 1. 申请列表筛选

#### 按状态筛选

**操作**: 点击统计卡片或使用下拉菜单

**状态类型**:
- **待审核** (黄色标签): 刚提交的申请，需要审核
- **已通过** (绿色标签): 已审核通过，商家账号已创建
- **已拒绝** (红色标签): 已审核拒绝，不予入驻

#### 关键词搜索

**支持搜索**:
- 店铺名称（模糊匹配）
- 联系人姓名（模糊匹配）
- 联系电话（模糊匹配）

**搜索技巧**:
- 输入部分关键词即可
- 搜索不区分大小写
- 按回车键快速搜索

### 2. 查看申请详情

#### 打开方式

- 方式 1: 点击"查看"按钮
- 方式 2: 双击表格行

#### 详情标签页

**基本信息**:
- 申请ID、申请时间
- 店铺名称、类型
- 联系人信息
- 登录账号

**资质信息**:
- 企业资质：企业名称、信用代码、法人、营业执照
- 个人资质：身份证号、身份证照片

**审批信息**:
- 审批状态、审批时间
- 审批人信息
- 审批意见
- 关联商家ID（通过后）

#### 证件照片查看

**操作**:
1. 点击证件图片
2. 系统打开大图预览
3. 可以缩放、拖动查看细节
4. 点击空白处或按 ESC 关闭

**审核要点**:
- ✅ 照片清晰度
- ✅ 证件完整性
- ✅ 有效期检查
- ✅ 信息一致性

### 3. 执行审批

#### 通过审核

**适用场景**:
- 资料齐全完整
- 证件真实有效
- 信息填写准确
- 符合平台入驻标准

**操作步骤**:
1. 点击"审批"按钮
2. 选择"通过审核"
3. （选填）填写审批意见
4. 点击"确认通过"
5. 确认二次提示
6. 完成审批

**系统自动操作**:
- 更新申请状态为"已通过"
- 创建商家账号
- 设置商家状态为"正常"
- 关联商家ID到申请记录
- 刷新列表和统计数据

#### 拒绝申请

**适用场景**:
- 证件不清晰或无效
- 信息填写不完整
- 资质不符合要求
- 店铺名称不合规
- 其他不符合入驻条件的情况

**操作步骤**:
1. 点击"审批"按钮
2. 选择"拒绝申请"
3. **必填**填写拒绝原因（至少5个字符）
4. 点击"确认拒绝"
5. 确认二次提示
6. 完成审批

**拒绝原因要求**:
- 必须明确指出问题所在
- 给出改进建议
- 语气专业、礼貌
- 至少5个字符

**系统自动操作**:
- 更新申请状态为"已拒绝"
- 记录拒绝原因
- 不创建商家账号
- 刷新列表和统计数据

## 常见问题

### Q1: 如何快速找到待审核的申请？

**A**: 
1. 方法 1: 点击顶部"待审核"统计卡片
2. 方法 2: 在筛选器中选择"待审核"状态
3. 方法 3: 按申请时间排序，最新的在最前面

### Q2: 审批后能否撤销？

**A**: 
当前版本不支持撤销审批。如需修改，请联系技术管理员直接操作数据库。
建议：审批前仔细核对信息，确认无误后再提交。

### Q3: 如何查看某个商家的申请记录？

**A**: 
1. 在申请列表页面使用搜索功能
2. 输入店铺名称或联系人信息
3. 查看搜索结果

### Q4: 审批意见能修改吗？

**A**: 
当前版本不支持修改审批意见。建议审批时认真填写。

### Q5: 批量审批功能在哪里？

**A**: 
当前版本暂不支持批量审批，需要逐个审核。
原因：确保每个申请都经过仔细审核，保证质量。

## 性能优化建议

### 大量申请处理

当待审核申请超过 100 条时：

1. **分批处理**: 每次处理 20-30 条
2. **优先级排序**: 先处理企业商家
3. **时间分配**: 每个申请审核时间控制在 2-3 分钟
4. **休息间隔**: 每处理 30 分钟休息 5 分钟

### 浏览器性能

- 定期清理浏览器缓存
- 关闭不必要的标签页
- 使用 Chrome 或 Edge 浏览器
- 确保网络连接稳定

## 注意事项

### ⚠️ 重要提醒

1. **审慎审批**: 审批决定会直接影响商家入驻，请仔细核对
2. **拒绝原因**: 拒绝时必须填写明确的原因，帮助商家改进
3. **证件真实性**: 注意识别PS或伪造的证件
4. **信息保密**: 商家的证件信息属于敏感数据，严格保密
5. **及时处理**: 避免申请长时间积压，影响商家体验

### 🔒 安全建议

1. 不要在公共场所审核申请
2. 审核完成后及时关闭页面
3. 定期修改管理员密码
4. 发现异常申请及时上报
