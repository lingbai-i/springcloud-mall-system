## 2. 系统技术栈

### 2.1 前端技术

#### 2.1.1 核心框架

**Vue.js 3.5.12**  
渐进式JavaScript框架，采用Composition API实现组件化开发。提供响应式数据绑定和虚拟DOM机制，具备高性能和灵活性。作为整个前端应用的基础框架，支撑所有页面和交互逻辑的实现。

**Vite 5.4.10**  
新一代前端构建工具，基于原生ES模块提供极速的开发体验。开发环境按需编译无需打包，生产环境使用Rollup优化打包。显著提升开发效率和构建性能，是项目的核心构建工具。

#### 2.1.2 UI组件库

**Element Plus 2.4.2**  
基于Vue 3的企业级UI组件库，提供60+高质量组件覆盖表单、表格、导航等场景。支持主题定制和响应式设计，是构建管理后台和用户界面的主要组件来源，统一视觉风格。

**ECharts 5.4.3**  
百度开源的数据可视化图表库，支持折线图、柱状图、饼图等多种图表类型。具备强大的配置能力和大数据渲染性能，用于商家销售统计、订单趋势分析等数据展示场景。

#### 2.1.3 路由与状态管理

**Vue Router 4.2.5**  
Vue官方路由管理器，支持嵌套路由、动态路由和路由懒加载。通过导航守卫实现权限控制，管理应用的页面导航和访问权限，是单页应用路由的核心实现。

**Pinia 2.1.7**  
Vue官方状态管理库，Vuex的下一代替代方案。提供简洁的API和模块化状态管理能力，支持TypeScript类型推断。管理用户信息、权限数据、购物车等全局状态，实现组件间数据共享。

**pinia-plugin-persistedstate 3.2.0**  
Pinia状态持久化插件，支持将状态自动保存到localStorage或sessionStorage。确保用户刷新页面后登录状态和购物车数据不丢失，提升用户体验。

#### 2.1.4 HTTP通信

**Axios 1.6.0**  
基于Promise的HTTP客户端，支持请求和响应拦截器。统一管理API请求、自动携带Token、处理错误响应。是前端与后端微服务通信的唯一网络库，封装所有HTTP请求逻辑。

#### 2.1.5 工具库

**@vueuse/core 10.5.0**  
Vue Composition API工具集合，提供200+可组合函数如防抖节流、窗口监听等。简化常用功能的实现，提高代码复用性和开发效率。

**dayjs 1.11.18**  
轻量级日期处理库（仅2KB），提供日期格式化、计算和解析功能。替代庞大的Moment.js，用于订单时间、统计区间等日期相关业务逻辑。

**nprogress 0.2.0**  
页面加载进度条库，在路由切换和异步请求时显示顶部进度条。提升用户体验，让用户感知页面加载状态。

#### 2.1.6 表单验证

**vee-validate 4.11.8 + yup 1.3.3**  
vee-validate是Vue表单验证库，yup是验证规则定义库。二者结合提供声明式表单验证能力，支持复杂验证逻辑。用于注册登录、商品发布等关键表单的数据校验。

#### 2.1.7 开发工具

**ESLint 8.53.0 + eslint-plugin-vue 9.18.1**  
JavaScript代码检查工具及Vue专用插件，统一代码规范和风格。自动发现潜在错误，保证代码质量和团队协作效率。

**Prettier 3.1.0**  
代码格式化工具，自动统一代码风格（缩进、引号、换行等）。与ESLint配合使用，确保代码库风格一致。

**Sass 1.69.5**  
CSS预处理器，支持变量、嵌套、混入等高级特性。提高样式代码的可维护性和复用性，简化复杂样式编写。

**unplugin-auto-import 0.16.7 + unplugin-vue-components 0.25.2**  
自动导入工具，无需手动import即可使用Vue API和组件。减少模板代码，提升开发体验和代码简洁度。

#### 2.1.8 测试工具

**Playwright 1.48.2**  
端到端自动化测试框架，支持多浏览器测试。用于测试关键业务流程（登录、下单、支付等），保证系统功能完整性和稳定性。

### 2.2 后端技术

#### 2.2.1 核心框架

**Java 17**  
Oracle官方长期支持（LTS）版本，提供优秀的性能和稳定性。支持记录类、文本块、模式匹配等现代语言特性，是项目后端开发的基础语言环境。

**Spring Boot 3.1.12**  
简化Spring应用开发的框架，提供自动配置和起步依赖。遵循约定优于配置原则，快速构建生产级应用。是所有微服务的基础框架，简化配置和部署流程。

**Spring Cloud 2022.0.4**  
微服务开发工具集，提供服务治理、配置管理、负载均衡等功能。基于Spring Boot构建分布式系统，统一微服务架构标准。

**Spring Cloud Alibaba 2022.0.0.0**  
阿里巴巴开源的微服务解决方案，整合Nacos、Sentinel等组件。提供完整的微服务生态和中文社区支持，是国内微服务架构的主流选择。

#### 2.2.2 微服务治理

**Nacos 2.3.0**  
阿里巴巴开源的服务注册与配置中心。实现服务注册发现、健康检查、动态配置管理。支持多环境命名空间隔离，是微服务架构的核心基础设施。

**Spring Cloud Gateway**  
基于Spring WebFlux的API网关，提供路由转发、负载均衡、限流鉴权等功能。作为系统统一入口，处理所有外部请求，实现安全控制和流量管理。

**OpenFeign**  
声明式HTTP客户端，简化服务间调用。通过接口和注解定义远程调用，自动集成负载均衡和熔断降级。是微服务间通信的主要方式。

**Sentinel 1.8.6**  
阿里巴巴开源的流量控制和熔断降级框架。提供实时监控、动态规则配置、系统自适应保护。防止服务雪崩，保障系统高可用性。

**Seata 1.7.0**  
阿里巴巴开源的分布式事务解决方案，支持AT、TCC、SAGA、XA多种模式。保证跨服务操作的数据一致性，解决分布式事务难题。

**RocketMQ 2.2.3**  
阿里巴巴开源的分布式消息中间件，支持事务消息、顺序消息、延时消息。用于订单、支付、库存等业务的异步处理和解耦，提升系统吞吐量。

#### 2.2.3 数据访问

**MySQL Connector 8.3.0**  
MySQL官方JDBC驱动，支持最新的MySQL 8特性。提供高性能的数据库连接和操作能力，是Java应用访问MySQL的标准驱动。

**MyBatis-Plus 3.5.7**  
MyBatis增强工具，无侵入式简化CRUD操作。提供代码生成器、分页插件、逻辑删除等实用功能。支持多租户和动态表名，大幅提升开发效率。

#### 2.2.4 缓存与存储

**Spring Data Redis 3.2.0**  
Spring对Redis的集成支持，提供RedisTemplate和ReactiveRedisTemplate。用于缓存热点数据、分布式锁、限流计数、会话管理。支持Redis Cluster集群模式。

**MinIO 8.5.12**  
高性能对象存储服务，兼容Amazon S3协议。用于存储商品图片、商家资质、用户头像等文件。支持本地部署和云端迁移，提供可靠的文件存储方案。

#### 2.2.5 工具库

**Hutool 5.8.32**  
Java工具类库，提供日期、字符串、集合、加密、HTTP等丰富工具方法。简化常用功能开发，避免重复造轮子，提高代码质量。

**FastJSON2 2.0.53**  
阿里巴巴JSON解析库，高性能低内存占用。支持Java对象与JSON的快速互转，用于API数据序列化和缓存数据处理。

**Lombok 1.18.32**  
Java代码简化工具，通过注解自动生成getter/setter、构造函数、日志对象等。减少样板代码，提升开发效率和代码可读性。

#### 2.2.6 API文档

**Knife4j 4.5.0**  
基于Swagger/OpenAPI 3.0的API文档工具，提供美观的UI界面。支持在线调试和网关聚合文档，方便前后端联调和接口管理。

#### 2.2.7 构建工具

**Maven 3.6+**  
项目构建和依赖管理工具，提供标准化的项目结构。通过POM文件管理依赖版本，支持多模块项目构建，是Java项目的标准构建工具。

### 2.3 数据库技术

#### 2.3.1 关系型数据库

**MySQL 8.0**  
开源关系型数据库管理系统，采用InnoDB存储引擎支持事务和外键。提供ACID特性保证数据一致性，支持JSON数据类型。按业务域分库设计（mall_user、mall_product、mall_order、mall_merchant等），存储核心业务数据。

#### 2.3.2 缓存数据库

**Redis 6.0+**  
高性能内存数据库，支持String、Hash、List、Set、ZSet等数据结构。用于热点数据缓存（商品、用户信息）、购物车存储、分布式锁、限流计数、Session管理。部署为Redis Cluster集群模式保证高可用。

#### 2.3.3 搜索引擎（预留）

**Elasticsearch 8.x**  
分布式搜索与分析引擎，支持全文检索和复杂查询。规划用于商品搜索、中文分词、模糊查询、聚合分析。通过独立的search-service提供搜索功能，提升用户检索体验。

## 3. 技术栈总结

### 3.1 前端技术特点

- **现代化**：采用Vue 3 + Vite最新技术栈
- **组件化**：Element Plus提供丰富UI组件
- **工程化**：ESLint + Prettier保证代码质量
- **高性能**：Pinia状态管理 + Axios网络请求优化

### 3.2 后端技术特点

- **微服务化**：Spring Cloud Alibaba完整微服务生态
- **高可用**：Sentinel限流熔断 + Seata分布式事务
- **易扩展**：服务独立部署，按需扩容
- **国产化**：Nacos、Sentinel、RocketMQ等国产组件

### 3.3 数据库技术特点

- **数据一致性**：MySQL事务保证核心数据安全
- **高性能**：Redis缓存加速热点数据访问
- **可扩展**：分库分表设计，支持水平扩展
- **多样化**：关系型 + 缓存 + 搜索引擎组合方案

## 4. 技术选型原则

### 4.1 成熟稳定

选择社区活跃、文档完善、经过生产验证的技术组件，降低技术风险。

### 4.2 生态完善

优先选择具有完整生态系统的技术栈，如Spring全家桶、Vue生态，便于集成和扩展。

### 4.3 性能优先

在保证功能的前提下，选择高性能的组件，如Vite构建工具、Redis缓存、FastJSON2序列化。

### 4.4 国产支持

积极采用阿里巴巴等国内厂商开源的优秀组件，获得中文社区和技术支持。
