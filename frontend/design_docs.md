# 国风商城首页设计说明文档

## 1. 设计理念 (Design Philosophy)

本次首页改版旨在打造一个融合传统东方美学与现代电商体验的"国风商城"。我们摒弃了传统的电商促销红，转而采用更具文化底蕴的色彩体系，营造出优雅、舒适的购物氛围。

### 核心视觉元素
- **色彩体系**：
  - **主色调 (Primary)**：竹青色 (`#5d7e68`) —— 象征自然、生机与雅致。
  - **辅助色 (Secondary)**：木色 (`#d4a373`) —— 传递温暖与质朴感。
  - **点缀色 (Accent)**：丹红 (`#b23a3a`) —— 用于价格和重要操作，保留传统韵味且醒目。
  - **背景色**：米宣纸色 (`#fdfbf5`) 搭配水墨山水背景图，增加空间层次感。

- **布局结构**：
  - 采用经典的"通栏导航 + 左侧分类 + 中间轮播 + 右侧功能区"的电商布局，确保用户上手成本低。
  - 使用半透明磨砂玻璃效果 (`backdrop-filter`) 让内容区域与水墨背景自然融合，体现现代设计感。

## 2. 技术实现细节 (Implementation Details)

### 2.1 响应式设计
- 使用 Flexbox 和 CSS Grid 布局系统。
- **桌面端**：完整显示分类侧边栏、轮播图和右侧用户信息区。
- **移动端/平板**：自动隐藏侧边栏和右侧信息区，商品网格从 4 列调整为 2 列，确保在小屏幕上的浏览体验。

### 2.2 性能优化
- **图片懒加载**：商品列表图片添加 `loading="lazy"` 属性，减少首屏加载流量。
- **CSS 优化**：使用 CSS 变量 (`:root`) 管理主题色，便于后续一键换肤。
- **资源引用**：直接引用本地优化后的静态资源，避免第三方 CDN 依赖。

### 2.3 交互体验
- **悬停效果**：商品卡片、按钮和导航项添加了平滑的 `transition` 过渡效果。
- **动态反馈**：搜索框、购物车等交互元素在聚焦或点击时有明确的视觉反馈。
- **无障碍性**：图片添加 `alt` 属性，保持语义化 HTML 结构。

## 3. 目录结构说明

- `src/views/Home.vue`: 核心首页组件，包含完整的布局和逻辑。
- `src/styles/index.scss`: 定义全局 CSS 变量和通用工具类。
- `public/主题/`: 存放所有国风主题相关的静态资源（图片、图标）。
